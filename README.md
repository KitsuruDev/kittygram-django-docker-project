# ОСНОВНЫЕ ШАГИ ИНИЦИАЛИЗАЦИИ:

1. Запулль весь проект;

2. Создай виртуальное окружение в корне проекта с именем venv (python -m venv venv);

3. Установи в окружение всё из requirements.txt (pip install -r requirements.txt) (пакеты и зависимости должны быть нормальными, но если будут какие-то проблемы - напиши в лс).
